Hierarchy
===

An abstract, strategic card game for two players

To run locally (Mac)
---

```sh
$ git clone git@github.com:NathanielWroblewski/hierarchy.git
$ cd hierarchy
$ python3 -m http.server && open http://localhost:8000
```
